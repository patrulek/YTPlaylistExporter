# YouTube Playlist Exporter

Important note: The script and this README were generated by AI.

## Overview

A Python script to **quickly export multiple YouTube playlists** into **one file** (CSV / TXT / Markdown).  
Works **without a YouTube API key** - it uses `yt-dlp`. Three performance modes: **flat**, **hybrid**, **full**.

## Features
- Input: multiple playlist URLs/IDs as arguments **or** a text file (`--list playlists.txt`).
- Output: a **single** file (`.csv`, `.txt`, `.md`) combining all playlists.
- Modes:
  - `flat` - fastest (title + URL + ID; channel/duration may be missing),
  - `hybrid` - fast listing + parallel enrichment of metadata,
  - `full` - most detailed (slower).
- `--dedupe` - remove duplicate videos across playlists (by video ID).
- Extra columns for CSV: `playlist_title`, `playlist_id`.

## Requirements
- Python 3.8+
- `yt-dlp`

```bash
pip install yt-dlp
```

## Quick start

```bash
# From a file with one playlist per line
python yt_playlist_exporter.py --list playlists.txt --mode flat -o out.md --format md
```

## Usage

```bash
python yt_playlist_exporter.py [PLAYLIST_URL ...] [options]
python yt_playlist_exporter.py --list playlists.txt [options]
```

Key options:
* o, --output PATH - output file path (defaults to playlists.csv/txt/md).
* format {csv,txt,md} - output format (default: csv).
* mode {flat,hybrid,full} - performance/metadata mode (default: full).
* workers N - number of threads in hybrid mode (default: 6).
* playlist-items RANGE - per-playlist range (e.g., 1-200, 1,5,10-20).
* cookies cookies.txt - cookies file for private/unlisted content.
* dedupe - remove duplicate videos by ID across all playlists.
* list FILE - file with playlist URLs/IDs (lines starting with # are ignored).


## Examples

```bash
# Multiple playlists as arguments, ultra fast (flat)
python yt_playlist_exporter.py URL1 URL2 URL3 --mode flat -o out.md --format md

# Hybrid: fast + metadata, with deduplication
python yt_playlist_exporter.py --list playlists.txt --mode hybrid --workers 8 --dedupe -o all.csv --format csv

# Full: rich metadata, but only first 200 items per playlist
python yt_playlist_exporter.py --list playlists.txt --mode full --playlist-items 1-200 -o part.csv
```

## Output format

* CSV columns:
playlist_title, playlist_id, index, title, channel, duration, url, id
* TXT/MD: single file with sections per playlist (headers). Original playlist order is preserved.

## Performance tips

* Fastest: --mode flat.
* Balanced speed & detail: --mode hybrid --workers 6-10.
* For very large playlists, use --playlist-items and export in chunks.
* --cookies can help speed/consistency (skips consent/age gates).
